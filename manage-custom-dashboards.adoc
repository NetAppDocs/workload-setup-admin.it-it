= Gestisci dashboard personalizzate
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Gestisci le tue dashboard personalizzate dal catalogo delle dashboard. Da lì puoi visualizzare i dettagli, aggiornare i dati, modificare le impostazioni della dashboard o eliminare le dashboard che non ti servono più.

.A proposito di questa attività
Tutti gli utenti del tuo account Workload Factory possono accedere a tutte le dashboard personalizzate. Ogni dashboard mostra dettagli di base come il nome del creatore, la data di creazione e se qualcuno la sta modificando al momento.

Le dashboard personalizzate sono di sola lettura. Non possono modificare le impostazioni di storage o del carico di lavoro. Puoi modificare la dashboard stessa, ma non le risorse che monitora.

.Prima di iniziare
Devi avere almeno una dashboard personalizzata nel tuo account Workload Factory.



== Accedi al catalogo della dashboard

.Fasi
. Accedere utilizzando uno dei link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["esperienze di console"^].
. Dal menu, seleziona *Panoramica dei carichi di lavoro* e poi *Dashboard personalizzate*.
+
Il catalogo delle dashboard mostra tutte le dashboard personalizzate nel tuo account Workload Factory.

. Esamina l'elenco del dashboard.




== Aggiorna i dati della dashboard

I dati del dashboard vengono caricati quando apri il dashboard. Puoi aggiornarlo per vedere le informazioni più recenti.

La dashboard ricarica i dati dai tuoi carichi di lavoro. L'ora dell'ultimo aggiornamento cambia per indicare quando i dati sono stati aggiornati.



== Modifica una dashboard

Puoi modificare una dashboard per cambiare ciò che mostra, incluse le metriche e i grafici.

.Fasi
. Nella pagina *Dashboard personalizzate*, individua la dashboard che vuoi modificare.
. Dal menu *Azioni* per quella dashboard, seleziona *Modifica dashboard*.
+
Se un altro utente sta modificando la dashboard, visualizzerai un messaggio come "Attualmente in fase di modifica da parte di [nome utente]". Attendi che abbia finito o coordinati con lui.

. Nella finestra di dialogo dell'IA, descrivi le modifiche che desideri apportare.
+
Questo processo è simile alla creazione di una nuova dashboard. Puoi modificare i carichi di lavoro, le metriche o gli intervalli di tempo che la dashboard tiene traccia.

. Rivedi le modifiche e seleziona **Salva**.


.Risultato
La dashboard si aggiorna con le tue modifiche. Anche la data dell'ultima modifica si aggiorna.



== Elimina una dashboard

Elimina le dashboard che non ti servono più. Questa operazione è permanente e non può essere annullata.

.Fasi
. Nella pagina *Custom dashboards*, individua la dashboard che desideri rimuovere.
. Dal menu a tendina *Azioni* di quella dashboard, seleziona *Elimina*.
. Conferma l'eliminazione.


.Risultato
La dashboard viene rimossa definitivamente dal tuo account Workload Factory e tutti gli utenti perdono l'accesso ad essa.
