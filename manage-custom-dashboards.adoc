= Gestisci dashboard personalizzate
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Gestisci le tue dashboard personalizzate dal catalogo delle dashboard. Da lì, puoi visualizzare i dettagli, aggiornare i dati o eliminare le dashboard che non ti servono più.

.A proposito di questa attività
Tutti gli utenti del tuo account Workload Factory possono accedere a tutte le dashboard personalizzate. Ogni dashboard mostra dettagli di base come il nome del creatore e la data di creazione.

Le dashboard personalizzate sono di sola lettura. Non possono modificare le impostazioni di storage o del carico di lavoro.

.Prima di iniziare
Devi avere almeno una dashboard personalizzata nel tuo account Workload Factory.



[[access-the-dashboard-catalog]]
== Accedi al catalogo della dashboard

.Fasi
. Accedi utilizzando uno dei link:console-experiences.html["esperienze di console"^].
. Dal menu, seleziona *Panoramica dei carichi di lavoro* e poi *Dashboard personalizzate*.
+
Il catalogo delle dashboard mostra tutte le dashboard personalizzate nel tuo account Workload Factory.

. Esamina l'elenco del dashboard.




[[refresh-dashboard-data]]
== Aggiorna i dati della dashboard

I dati del dashboard vengono caricati quando apri il dashboard. Puoi aggiornarlo per vedere le informazioni più recenti.

La dashboard ricarica i dati dai tuoi carichi di lavoro. L'ora dell'ultimo aggiornamento cambia per indicare quando i dati sono stati aggiornati.



[[delete-a-dashboard]]
== Elimina una dashboard

Elimina le dashboard che non ti servono più. Questa operazione è permanente e non può essere annullata.

.Fasi
. Nella pagina *Custom dashboards*, individua la dashboard che desideri rimuovere.
. Dal menu a tendina *Azioni* di quella dashboard, seleziona *Elimina*.
. Conferma l'eliminazione.


.Risultato
La dashboard viene rimossa definitivamente dal tuo account Workload Factory e tutti gli utenti perdono l'accesso ad essa.
